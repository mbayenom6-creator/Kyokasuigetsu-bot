# 🌙 Kyokasigetsu Bot

Un bot performant pour WhatsApp conçu pour automatiser vos tâches et divertir vos groupes.

## 🚀 Fonctionnalités
* **Commandes de groupe :** Gestion des membres, tagall, etc.
* **Multimédia :** Téléchargement de vidéos et d'images.
* **Interaction :** Réponses automatiques et jeux.
* **Stabilité :** Déployé 24/7 sur Railway.

## 🛠️ Installation & Déploiement

### Déploiement sur Railway
1. Forkez ce dépôt.
2. Connectez votre compte GitHub à [Railway](https://railway.app/).
3. Créez un nouveau projet à partir du dépôt.
4. Ajoutez votre fichier `railway.json` (déjà fait ! ✅).

### Configuration
Pour connecter le bot à WhatsApp, vous devrez généralement scanner un QR Code dans les logs ou configurer une `SESSION_ID`.

## ⚙️ Variables d'Environnement
Voici les variables à configurer dans Railway pour le bon fonctionnement :
* `OWNER_NUMBER` : Votre numéro de téléphone.
* `PREFIX` : Le symbole pour appeler le bot (ex: `!`).
